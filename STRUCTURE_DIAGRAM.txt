UI FLOW
=======

USER NAVIGATES TO /admin/manage
│
├─ Sees Sidebar with Categories
│  ├─ Click category → Expands subcategories
│  ├─ Click subcategory → Highlights it
│  └─ Click "+ Add Category" → Opens form modal
│
├─ Sees Services Panel (auto-populated from database)
│  ├─ Shows all services with details
│  ├─ Shows subservices and questions count
│  └─ "+ Add Service" button → Opens form modal
│
├─ Sees Officers Panel (auto-populated from database)
│  ├─ Shows all officers with full details
│  ├─ Email, phone, specialization visible
│  └─ "+ Add Officer" button → Opens form modal
│
└─ Sees Ads Panel (auto-populated from database)
   ├─ Shows all ads with priority and dates
   ├─ Type and target audience visible
   └─ "+ Add Advertisement" button → Opens form modal


API ENDPOINTS
=============

GET  /api/admin/categories              → List all categories with subcategories
POST /api/admin/categories              → Create/Update category
DELETE /api/admin/categories?id=...     → Delete category

POST /api/admin/categories/add-subcategory  → Add subcategory to category

GET  /api/admin/services                → List all services
POST /api/admin/services                → Create/Update service
DELETE /api/admin/services/{id}         → Delete service

GET  /api/admin/officers                → List all officers
POST /api/admin/officers                → Create/Update officer
DELETE /api/admin/officers?id=...       → Delete officer

GET  /api/admin/ads                     → List all ads
POST /api/admin/ads                     → Create/Update ad
DELETE /api/admin/ads?id=...            → Delete ad
