SCENARIO 1: Viewing Category Details
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  User clicks category → Category highlights
         ↓
  User clicks ▶ arrow → Expands subcategories
         ↓
  Shows all subcategories with item counts
         ↓
  User can click "+ Add Subcategory" → Opens form

SCENARIO 2: Adding New Content
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  User clicks "+ Add" button → Form modal opens
         ↓
  Fills all required fields (marked with *)
         ↓
  Clicks "Save" → Form submits to API
         ↓
  API validates and saves to MongoDB
         ↓
  Success response → Page reloads
         ↓
  New item appears in list

SCENARIO 3: Editing Existing Content
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  User finds existing item → Clicks "Delete"
         ↓
  Confirms deletion → Sends DELETE request
         ↓
  Item removed from database
         ↓
  List updates to reflect change

  • Description         (optional)
  • Contact Email       (optional)
  • Hotline             (optional)
  • Website             (optional)
